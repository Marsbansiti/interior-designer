## 📋 Overview

**Interior Designer** is an AI Studio web application that leverages Google's Gemini API to provide intelligent interior design recommendations and assistance. The application runs locally and can be deployed to production environments.

**Live Application:** https://ai.studio/apps/4154bfbb-d9ea-4f51-bf5e-e7f0cdb0357e

## 🎯 Project Purpose

This project serves as a complete, ready-to-run interior design application powered by AI. Users can interact with the app through a web interface to:
- Get interior design suggestions and ideas
- Receive AI-powered design recommendations
- Explore creative solutions for space planning
- Access professional interior design insights via Gemini AI

## 🚀 Getting Started

### Prerequisites
Ensure you have Node.js installed on your system.

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Marsbansiti/interior-designer.git
   cd interior-designer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment variables**
   - Open `.env.local` file
   - Set your Gemini API key:
     ```
     GEMINI_API_KEY=your_api_key_here
     ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Access the application**
   - Open your browser and navigate to the provided local URL (typically `http://localhost:3000`)
